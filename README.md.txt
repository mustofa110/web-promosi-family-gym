# Web-Based Promotion System - Family GYM Kotabumi

Repositori ini berisi kode sumber untuk sistem informasi dan media promosi berbasis web yang dirancang khusus untuk **Family GYM Kabupaten Lampung Utara**. Projek ini merupakan hasil implementasi dari penelitian akademis untuk mendigitalisasi proses promosi yang sebelumnya masih dilakukan secara manual.

Projek ini bertujuan untuk meningkatkan jangkauan pemasaran dan efisiensi pengelolaan informasi layanan di Family GYM. Sistem ini memungkinkan calon member untuk melihat fasilitas, jadwal, dan layanan secara online, serta memudahkan pengelola dalam manajemen data operasional.
